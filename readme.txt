HammerPress is a free WordPress theme for construction and home improvement related websites. It's great for business and/or blogging purposes and is incredibly easy to set up.

WP Customize Options Include:
-Site Name, Description, Logo, Site Icon (Favicon)
-Background Color and Background Image
-Primary Color

Sidebar / Widget Areas
-Left Side Sticky Sidebar
-Home - Left Column Widget
-Home - Right Column Widget
-Above Post - for ad banners, Adsense, etc.
-Below Post - another place for ads
